Lumen IM 后端

## 目录结构说明

```
    app/                    应用目录
       cache/               缓存处理
       http/                http服务
            handler/        handler 处理
            middleware/     中间件
            response/       响应
            request/        请求
            router/         路由
       model/               model定义
       dao/                 dao定义
       service/             服务层 
       pkg/                 包
   config/                  配置文件
   resource/                资源目录
   runtime/                 运行目录，存放日志
```
